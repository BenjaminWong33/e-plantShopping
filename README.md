# e-plantShopping

A React + Redux Toolkit e-commerce plant shopping application.

## Features

- Add plants to cart
- Update quantity
- Remove products
- Global state management using Redux Toolkit
- Responsive shopping cart

## Technologies Used

- React
- Redux Toolkit
- React Redux
- CSS

## Redux Features

- configureStore()
- createSlice()
- useDispatch()
- useSelector()
